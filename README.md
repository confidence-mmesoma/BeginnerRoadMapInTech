# BeginnerRoadMapInTech
Features
8 Curated Roadmaps: Frontend, Backend, AI, Data Science, and more.

The "One Source" Rule: Each path points to a single, trusted educator to prevent tutorial hell.

Persistent Progress: Automatically saves your status using localStorage.

Visual Feedback: Real-time progress bar and "Journey Complete" celebrations.

Clean UI: Built for focus with a calm, mobile-friendly design.

Tech Stack
HTML5

CSS3 (Custom variables & animations)

JavaScript (Vanilla JS, DOM Manipulation, LocalStorage)

 File Structure
index.html: The core layout and selection menu.

style.css: Clean, modern styling and celebratory animations.

script.js: Handles data rendering, progress logic, and browser memory.

 How to Use
Clone or download the folder.

Open index.html in any browser.

Select a learning path.

Check topics as you learn. Your progress saves automatically!
frontend project, built with html, css and javaScript.
